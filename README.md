# Assignment-Page
This is the page for the round-2 assignment for Deep Thought recruitment

## Important note:
To run the page, follow the below steps:
- Open root folder in VS code
- Use extension 'Live Preview' to run the page locally
- Open index.html and click on 'Show preview'
- On the preview tab, click on hamburger icon to open option, and click on open in browser.
- Page will be live at `http://127.0.0.1:3000/index.html`
